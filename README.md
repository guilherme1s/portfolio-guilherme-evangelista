# Portfólio Guilherme Evangelista

Aplicação web desenvolvida para apresentação profissional, destacando experiências, projetos e habilidades técnicas de forma moderna e responsiva.

## Funcionalidades

- Estrutura organizada em cinco seções: Sobre mim, Experiências, Projetos, Habilidades e Contato
- Navegação fluida entre seções com rolagem suave
- Sistema de tema claro e escuro
- Layout totalmente responsivo para desktop, tablet e mobile
- Listagem de projetos com destaque para resultados e tecnologias utilizadas
- Formulário de contato funcional com envio de mensagens via EmailJS

## Decisões Técnicas
- Arquitetura baseada em componentes reutilizáveis para facilitar manutenção e escalabilidade
- React + TypeScript para tipagem estática e maior segurança no desenvolvimento
- Tailwind CSS para estilização utilitária com foco em produtividade e consistência visual
- Organização do projeto por responsabilidade (components, sections, hooks, services)
- Deploy automatizado utilizando Vercel

## Tecnologias
- React
- TypeScript
- Tailwind CSS
- EmailJS
